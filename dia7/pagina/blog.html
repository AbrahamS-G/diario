Como cree la pagina?




  1. Primer paso

    • Mejoramos el login creando paginas de registrar y olvide mi contraseña

    • Agregamos JavaScript para simular un logueo o un registro

    • Creamos la pagina principal llamada 'inicio'

  2. Segundo paso

    • Linkeamos el login con 'inicio' para quee cuando acceda, lo redirija

    • Creamos las sub-paginas para las demas secciones.
      'proyectos' 'pedidos' 'contacto'

    • Le damos estilo al 'header' 'main' y 'footer'

  3. Tercer paso

    • Se unen las paginas para que esten entrelazadas

    • Agregamos contenido a 'inicio'

    • Agregamos contenido a 'pedidos'

    • Se hace el formulario de 'contacto'

    • Se linkean los proyectos

  4. Cuarto y ultimo paso

    • Se agrega el js para los cursos, poder inscribirse o no

    • Crear el js del formulario de 'Contacto'

    • Se acomodan estilos para mejor presentacion

    • Se crea el blog de como hice la pagina porque ya no sabia que mas agregarle

    • Se termina otro proyecto y se agrega al portfolio
